À Quatre Mains
experience here

À Quatre Mains is a musical instrument in VR and requires tracked hand controls.

It can be explored without VR via mouse however it is a limited experience.

Controls

  • Touch a node to activate it temporarily
  • Touch and hit trigger button to lock that node to active.
  • Touch and hit grab button to active and lock that node as an arpeggiator.

Multi user version coming soon.

Appearances

(June, 2017) À Quatre Mains is revealed at 3D Web Fest
